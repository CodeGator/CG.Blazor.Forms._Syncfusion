    Class RenderSfNumericTextBoxAttribute

    This class is an attribute that, when applied to a numeric property, causes the form generator to render the property as a Syncfusion.Blazor.Inputs.SfNumericTextBox`1 component.

    Syntax
    [AttributeUsage(AttributeTargets.Property)]
public class RenderSfNumericTextBoxAttribute : SyncfusionAttribute
    Remarks

    This attribute is only valid when placed on a property of type: numeric, which is byte, int, long, float, double, or decimal.

    Examples

    Here is an example of decorating a model property to render a Syncfusion.Blazor.Inputs.SfNumericTextBox`1:

    using CG.Blazor.Forms.Attributes;
class MyModel
{
    [RenderSfNumericTextBox]
    public byte MyProperty { get;set; }
}
