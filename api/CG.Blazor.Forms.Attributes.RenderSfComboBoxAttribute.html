    Class RenderSfComboBoxAttribute

    This class is an attribute that, when applied to a System.String property, causes the form generator to render the property as a Syncfusion.Blazor.DropDowns.SfComboBox`2 component.

    Syntax
    [AttributeUsage(AttributeTargets.Property)]
public class RenderSfComboBoxAttribute : SyncfusionAttribute
    Remarks

    This attribute is only valid when placed on a property of type: System.String.

    Examples

    Here is an example of decorating a model property to render a Syncfusion.Blazor.DropDowns.SfComboBox`2:

    using CG.Blazor.Forms.Attributes;
class MyModel
{
    [RenderSfComboBox(Options = "A,B,C,D")]
    public string MyProperty { get;set; }
}
