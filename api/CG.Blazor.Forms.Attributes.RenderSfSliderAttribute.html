    Class RenderSfSliderAttribute

    This class is an attribute that, when applied to a numeric property, causes the form generator to render the property as a Syncfusion.Blazor.Inputs.SfSlider`1 component.

    Syntax
    [AttributeUsage(AttributeTargets.Property)]
public class RenderSfSliderAttribute : SyncfusionAttribute
    Remarks

    This attribute is only valid when placed on a property of type: numeric, which is byte, int, long, float, double, or decimal.

    Examples

    Here is an example of decorating a model property to render a Syncfusion.Blazor.Inputs.SfSlider`1:

    using CG.Blazor.Forms.Attributes;
class MyModel
{
    [RenderSfSlider]
    public int MyProperty { get;set; }
}
