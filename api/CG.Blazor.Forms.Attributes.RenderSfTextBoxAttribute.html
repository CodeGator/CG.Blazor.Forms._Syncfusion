    Class RenderSfTextBoxAttribute

    This class is an attribute that, when applied to a System.String property, causes the form generator to render the property as a Syncfusion.Blazor.Inputs.SfTextBox component.

    Syntax
    [AttributeUsage(AttributeTargets.Property)]
public class RenderSfTextBoxAttribute : SyncfusionAttribute
    Remarks

    This attribute is only valid when placed on a property of type: System.String.

    Examples

    Here is an example of decorating a model property to render a Syncfusion.Blazor.Inputs.SfTextBox:

    using CG.Blazor.Forms.Attributes;
class MyModel
{
    [RenderSfTextBox]
    public string MyProperty { get;set; }
}
