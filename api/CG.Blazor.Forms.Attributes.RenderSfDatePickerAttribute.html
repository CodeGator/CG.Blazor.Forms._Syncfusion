    Class RenderSfDatePickerAttribute

    This class is an attribute that, when applied to a property, causes the form generator to render the property as a Syncfusion.Blazor.Calendars.SfDatePicker`1 component.

    Remarks

    This attribute is only valid when placed on a property of type: .

    Examples

    Here is an example of decorating a model property to render a Syncfusion.Blazor.Calendars.SfDatePicker`1:

    using CG.Blazor.Forms.Attributes;
class MyModel
{
    [RenderSfDatePicker]
    public DateTime? MyProperty { get;set; }
}
